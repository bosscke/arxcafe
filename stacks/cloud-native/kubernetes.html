K8s

Kubernetes

Container Orchestrator
  • Pods, Services, Deployments, StatefulSets for resilient workloads
  • Ingress controllers for routing, service discovery
  • Auto-scaling, rolling updates, self-healing health checks
↓
D

Docker

Container Runtime
  • Image builds from Dockerfiles, layered image cache
  • Container isolation, resource limits, networking
  • Registry push/pull (Docker Hub, ECR, GCR, ACR)
↓
H

Helm

Kubernetes Package Manager
  • Charts with templates and values.yaml for reusable releases
  • Versioning, rollbacks, dependency management
  • Declarative installs across environments
Flow: Helm charts configure releases → Kubernetes schedules pods/services → Docker runtime executes containers across nodes.
